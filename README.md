# WikiSearch

Wikipedia Search Android native Kotlin App
This app is used to search Wikipedia with text and Voice input as well.

 **Components Used:**   
 - Kotlin   
 - MVVM   
 - Data Binding   
 - Speech Recognizer  
 - Live Data   
 - Jetpack Navigation    
 - Glide   
 - Lottie Animations   
 - ROOM Database   
 - Coroutines   
 - Dexter    
 - Retrofit    
 - GSON
